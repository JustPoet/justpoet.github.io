插入排序

时间复杂度O(n^2), 空间复杂度O(1)

思路：

  1. 将数据划分为两个区域，已排序区和未排序区，最开始已排序区为第一位数字，从第二位数字(第一个未排序数字)开始检测；
  2. 比较后移。将当前未排序的数据，在已排序区从后往前与已排比较，如果已排序区的数据大则后移一位并继续进行前一位的比较。
  3. 插入。当前序数据小于或者已查找完所有已排序区数据时，将待排数据插入到索引位置。
